# `Swap` function exposes users to unlimited slippage

## Summary

Unitas `swap` provides no slippage protection, that will expose users to unlimited slippage.

## Vulnerability Detail

The `swap` function in Unitas is used to swap tokens, however, users cannot specify the max token amount they are willing to spend or the min token amount they expect to get. Thereâ€™s extreme situations when user's transaction stuck in pending and a large price fluctuation occurs before the transaction executes, this will exposes users to unlimited slippage.
